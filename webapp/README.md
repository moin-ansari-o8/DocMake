# DocMake Webapp

This webapp is implemented from `WEBAPP_BLUEPRINT.md` with a professional monochrome theme (black, white, gray shades only).

## Frontend

```bash
cd /home/runner/work/DocMake/DocMake/webapp/frontend
npm install
npm run dev
```

## Backend

```bash
cd /home/runner/work/DocMake/DocMake/webapp/backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
```

Frontend: `http://localhost:5173`
Backend: `http://localhost:8000`
